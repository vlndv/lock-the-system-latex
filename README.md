This repository contains a **CV-ready, executive LaTeX version** of the paper *Lock The System: A Structural Liquidity Framework for Intraday XAU/USD* by **Valentino De Vivo**.

## Contents
- `main.tex`: LaTeX source (A4, 1-inch margins, executive summary style)
- Embedded **TikZ** diagram for the Structural Hierarchy section
- Embedded **PGFPlots** chart for the Time-Based Volatility section

## How to Compile
1. Open [Overleaf](https://www.overleaf.com/) and upload `main.tex`, or
2. Compile locally with `pdflatex` (ensure `tikz` and `pgfplots` are installed):
   ```bash
   pdflatex main.tex
   ```
